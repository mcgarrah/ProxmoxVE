# Proxmox VE Helper-Scripts

## Project Overview

Proxmox VE Helper-Scripts is a comprehensive collection of community-driven automation scripts designed to simplify the setup and management of Proxmox VE environments. Originally created by tteck and now maintained by the community, this project provides one-command installations for popular services and containers.

## Key Features

### Quick Setup & Automation
- **One-Command Installations**: Simple bash commands for deploying 400+ popular services and containers
- **Interactive Installation**: User-friendly prompts with simple mode for beginners and advanced options for power users
- **Automated Container Creation**: Streamlined LXC container provisioning with optimal configurations

### Comprehensive Service Support
- **Self-Hosted Applications**: Home Assistant, Jellyfin, Nextcloud, Plex, and hundreds more
- **Development Tools**: Docker, Gitea, Jenkins, Code Server, and development environments
- **Network Services**: Pi-hole, AdGuard, Traefik, Nginx Proxy Manager, VPN solutions
- **Database Systems**: MariaDB, PostgreSQL, MongoDB, Redis, and specialized databases
- **Monitoring & Analytics**: Grafana, Prometheus, Zabbix, Uptime Kuma, and observability tools

### Management & Maintenance
- **Auto-Update Mechanisms**: Built-in update scripts for keeping installations current
- **Post-Install Configuration**: Automated setup scripts for optimal service configuration
- **Backup & Migration**: Tools for data backup and service migration between containers
- **Resource Management**: Intelligent resource allocation and optimization

### Multi-Platform Support
- **LXC Containers**: Debian and Alpine-based container installations
- **Virtual Machines**: Pre-configured VM deployments for specialized operating systems
- **Hybrid Deployments**: Support for both containerized and virtualized workloads
